# Hardware Architecture

## Gateway Hardware

- Raspberry Pi 4 or Raspberry Pi 5  
- external SSD recommended  
- official touchscreen display  
- cooling solution such as heatsink or fan  

---

## Sensor Node Hardware

- ESP32 microcontroller  
- connected environmental sensors  
- optional LoRa module for long range communication  

---

## Physical Layout

- sensor nodes distributed across environment  
- gateway placed centrally for connectivity  
- touchscreen attached to gateway for local interaction
